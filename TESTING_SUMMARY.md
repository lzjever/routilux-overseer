## ⚠️ 发现的问题和修复

### 问题 1: API 类未导出

**错误信息**:
```
Type error: Module '"@/lib/api"' declares 'FlowsAPI' locally,
but it is not exported.
```

**原因**: `lib/api/index.ts` 未导出插件所需的 API 子类型。

**修复**: 插件使用 `API["flows"]` 等（见 `lib/plugins/types.ts`）。`createAPI` 与 `API` 类型从 `@/lib/api` 导出。

**状态**: ✅ 已修复
